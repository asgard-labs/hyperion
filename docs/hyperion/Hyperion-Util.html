retryRepeated n doTry m tries to run doTry m n-1 times, after which it runs m 1 time. After each failure waits 15-90 seconds randomly. Returns on first success. Failure is represented by a Left value.

shellEsc :: FilePath -> [String] -> String #

Takes a path and a list of String arguments, shell-escapes the arguments, and combines everything into a single string.

onceWithArgument :: (MonadIO m, MonadIO n) => (m a -> n b) -> m a -> n (a, b) #

Given a function that takes a monadic action, allow us to inspect the argument to that function. Only works if f is guaranteed to evaluate its argument exactly once. This function is useful in combination with remoteBind, since it allows one to inspect the argument produced by the monadic action passed to the given continuation.
